# Machine-Learning-Based-Port-Scanning-Detection
Machine Learning-based network security system for detecting port scanning attacks using traffic analysis and classification algorithms. Machine Learning-Based Port Scanning Detection using Python and Scikit-learn to identify suspicious network scanning activities and improve network security.

Overview:-
This project uses Machine Learning techniques to detect port scanning activities in network traffic. Port scanning is a common reconnaissance technique used by attackers to identify open ports and vulnerable services on a target system. The model analyzes network traffic features and classifies traffic as normal or suspicious.

Features:-
Network traffic data preprocessing and cleaning
Feature extraction and selection
Machine Learning-based attack detection
Classification of normal and malicious traffic
Performance evaluation using standard metrics
Early detection of potential reconnaissance attacks

Technologies Used:-
Python
Scikit-learn
Pandas
NumPy
Matplotlib

Project Workflow:-
Data Collection--->Data Preprocessing--->Feature Engineering--->Model Training--->Model Testing--->Performance Evaluation--->Attack Detection

Dataset Info:-
Dataset Name: CICIDS2017
Contains network traffic records labeled as normal or attack traffic.
Used for training and testing the machine learning model.
